# SmartCam-ESP32

## 📌 Project Overview
SmartCam is an IoT-based security system using ESP32-S3, PIR sensor, and camera module.

It detects motion and sends a real-time alert with image to Telegram.

---

## 🔧 Features
- Motion detection using PIR sensor
- Image capture using camera module
- Telegram alert system
- Wi-Fi + AP setup configuration
- User login security
- Local access via smartcam.local
